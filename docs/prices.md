# Api для получения цены на хранение

### Url and methods

> **Url**: /price/info

> **Method**: POST

### Данные для получения цены

1. fileSize - размер файла в байтах

### Пример

###### Запрос

```
 http://localhost:3000/price/info
```

Тело запроса:
```
{
	"fileSize": "153132131321"
}
```

###### Ответ
```
{
	"amount": "0.6546546546"
}
```
